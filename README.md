## Features
    - Optimize images from your local folder by using a simple command
    - redim convert your images to webp

## How to use
It's very simple and easy to use.

    1. Copy the entire path of the image from your folder
    2. Go to the command palette and type redim quality : and select the quality you want to use for the image.
    3. Then paste the path into the input and the info box will show you the link to the optimized image.
    4. Ready! paste de link within your file.

**Tip: Usign redim quality : 90 reduce the size of your image without losing quality.**

You can save the link and use it within your html tags instead of uploading the image from your local, and the image  is hosted by cloudinary.

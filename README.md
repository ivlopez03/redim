## Features
It's very simple and easy to use.

    1. Copy the path of the image to be optimized.
    2. Go to the command palette and type redim quality : and select the quality you want to use for the image.
    3. Then paste the path into the input and the info box will show you the link to the optimized image.

You can save the link and use it in your html tags instead of uploading the image from your local, and the image  is hosted by cloudinary.
